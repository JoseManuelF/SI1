Para ejecutar el script crearDB.sh hay que incluir el dump de la base de datos
"dump_v1.2-P3" en las carpetas correspondientes.

- En el ejercicio 1 se debe incluir en la ruta Ejercicio 1_NoSQL/funciones/sql/
- En el ejercicio 3 se debe incluir en la ruta Ejercicio 3_Transacciones/

--------------------------------------------------------
En el ejercicio 1 en la ruta Ejercicio 1_NoSQL/funciones/ está el archivo crear.sh
que crea la base de datos de PostgreSQL, ejecuta el script createMongoDBFromPostgreSQLDB.py,
y hace las actualizaciones necesarias de la BD para que la página web funcione.
